# 💖 AmoraScribe AI

![AmoraScribe Logo](public/logo.png)

### *Write with Elegance. Correct with Intelligence.*

**AmoraScribe** is a premium, browser-based AI autocorrect application designed to elevate your writing experience. Combining a sophisticated, heart-inspired aesthetic with robust Python-powered linguistic processing, AmoraScribe ensures your words are as beautiful as your thoughts.

---

## ✨ Key Features

-   **🧠 In-Browser Python Intelligence**: Leveraging **Pyodide**, AmoraScribe runs complex Python linguistic logic directly in your browser, ensuring privacy and lightning-fast processing.
-   **✍️ Contextual Grammar Correction**: Beyond simple spell-checking, our AI understands verb tenses and grammatical nuances (e.g., automatically correcting "I has" to "I have").
-   **✨ Real-time Suggestions**: Get instant, elegant suggestions as you type. Our unobtrusive "suggestion chips" allow for one-click refinements.
-   **💎 Premium Glassmorphic UI**: A state-of-the-art interface featuring smooth gradients, subtle micro-animations, and a refined color palette (`#ff4d6d`, `#c9184a`).
-   **📊 Live Writing Stats**: Keep track of your word and character counts in real-time with our minimal status bar.
-   **🚀 Instant Engine Feedback**: A dedicated status indicator keeps you informed about the AI engine's initialization and activity.

---

## 🛠️ Technology Stack

-   **Frontend**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
-   **AI Engine**: [Pyodide](https://pyodide.org/) (Python in WebAssembly)
-   **Styling**: Vanilla CSS with Modern Design Tokens
-   **Deployment**: Optimized for [Netlify](https://www.netlify.com/)

## 🚀 Getting Started

### Prerequisites

-   [Node.js](https://nodejs.org/) (v18 or higher)
-   [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/amorascribe.git
    cd amorascribe
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open your browser**:
    Navigate to `http://localhost:5173` to start writing with elegance.

## 🎨 Design Philosophy

AmoraScribe is built on the principle that writing tools should be as inspiring as the writing itself. We use:
-   **Typography**: *Outfit* for modern readability and *Playfair Display* for classic elegance.
-   **Color Palette**: A curated selection of pinks and deep reds to evoke passion and precision.
-   **Interactivity**: Subtle hover effects and scale transitions to make the interface feel alive.
